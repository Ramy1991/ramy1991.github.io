# My Professional Portfolio

This folder contains the production-ready files for my portfolio website, prepared for deployment on GitHub Pages.

## Deployment Instructions

1.  **Create a Repository**: Create a new repository on GitHub (e.g., `ramygharib.github.io`).
2.  **Upload Files**: Upload the contents of this folder (`index.html`, `style.css`, etc.) to the root of that repository.
3.  **Enable GitHub Pages**:
    *   Go to your repository **Settings**.
    *   Click on **Pages** in the left sidebar.
    *   Under **Build and deployment**, ensure the source is set to "Deploy from a branch" and select the `main` (or `master`) branch.
4.  **View Your Site**: Your site will be live at `https://<yourusername>.github.io/`.

## Files Included

- `index.html`: The main structure of the website.
- `style.css`: Modern glassmorphism design and styles.
- `script.js`: Interactive elements and theme toggling.
- `Ramy_Gharib_Resume.pdf`: Professional resume.
